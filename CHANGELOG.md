# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-25

### Added
- Initial release
- Job scraping for E-INFRA Group (Electrogrup, Direct One, Netcity, Nova Power & Gas, WESEE)
- Python implementation using BeautifulSoup4 and Requests
- Solr integration for job indexing
- GitHub Actions workflow for daily automated scraping
- Integration tests for Solr connection and data integrity
- Support for multiple companies within the same group
